# قواعد السلوك

هدف Asas OS أن يكون مكانًا هادئًا وتقنيًا ومرحبًا لبناء نظام تشغيل بشكل جماعي.

## السلوك المتوقع

- الاحترام والصبر.
- افتراض حسن النية، خصوصًا مع المساهمين الجدد.
- نقد الكود والأفكار، وليس الأشخاص.
- شرح المفاضلات بوضوح عند مراجعة أجزاء معقدة مثل kernel أو filesystem.
- الحفاظ على تركيز النقاشات على المشروع.

## السلوك غير المقبول

- المضايقة أو التهديد أو الإهانة أو الهجوم الشخصي.
- التمييز بسبب الهوية أو الخلفية أو اللغة أو مستوى الخبرة.
- نشر معلومات خاصة بدون إذن.
- تعطيل النقاشات التقنية بشكل متكرر.
- الضغط على المشرفين لدمج تغييرات غير آمنة في التخزين أو أنظمة الملفات أو الأمان.

## التنفيذ

يمكن للمشرفين تعديل أو حذف تعليقات، إغلاق Issues، رفض Pull Requests، أو حظر
مشاركين إذا كان السلوك يضر بالمجتمع.

للإبلاغ عن مشكلة سلوك، تواصل مع المشرفين بشكل خاص. إذا لم توجد قناة خاصة بعد،
افتح Issue مختصرة تطلب قناة تواصل خاصة بدون نشر تفاصيل حساسة.
